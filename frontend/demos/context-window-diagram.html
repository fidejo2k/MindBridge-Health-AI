‚ö° The Core Insight
Claude itself has no memory between sessions. It is stateless. It only knows what you put in the context window right now. This means every behavior difference between a clinician and a patient ‚Äî the tone, what data is visible, what actions are allowed ‚Äî is entirely a result of what your backend injects at session start. Context engineering is access control. In healthcare, this is also HIPAA compliance.
